# Schema

A **schema** defines the structure and content of a stream of JSON values. Go to the [schema concept page](/docs/concepts/schema) for an in-depth explanation.

## Primitive

All effects schemas and custom schema types are ultimately defined with the following set of primitive schema types:

- [array](/docs/schema/primitive/array)
- [constant](/docs/schema/primitive/constant)
- [context](/docs/schema/primitive/context)
- [function](/docs/schema/primitive/function)
- [number](/docs/schema/primitive/number)
- [object](/docs/schema/primitive/object)
- [reference](/docs/schema/primitive/reference)
- [select](/docs/schema/primitive/select)
- [string](/docs/schema/primitive/string)

## Custom

See [custom schema types](/docs/concepts/schema#custom-schema-types) to learn how to make your own schema types.
