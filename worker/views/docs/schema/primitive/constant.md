# constant

Endlessly emits the specified literal value.

## Inputs

### `value` — required

The JSON value to be emitted.

## Examples

String:

```json
{
  "type": "constant",
  "value": "Avagadro"
}
```

Object:

```json
{
  "type": "constant",
  "value": {
    "name": "Évariste Galois",
    "age": 20
  }
}
```
